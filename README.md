# Job Tracker

A simple job application tracker built with Next.js.

## What it does

- Add job applications with company, role, location, salary
- Track status: Applied → Interview → Offer → Rejected
- Delete applications
- Data saved in browser localStorage

## Tech

- Next.js 14
- TypeScript
- Tailwind CSS
- Lucide icons

## Run

```bash
npm install
npm run dev
```
